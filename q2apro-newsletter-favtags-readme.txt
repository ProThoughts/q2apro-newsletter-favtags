# q2apro Premium Plugin: Newsletter FavTags

# Description

	Users can subscribe to their favorite tags and the plugin emails them daily if there are new questions.


# Installation

    - Buy the plugin and download the ZIP file provided q2apro-newsletter-favtags.zip
    - Make a full backup of your q2a database before installing the plugin.
    - Extract the folder q2apro-newsletter-favtags from the ZIP file.
    - Move the folder q2apro-newsletter-favtags to the qa-plugin folder of your Q2A installation.
    - Use your FTP-Client to upload the folder q2apro-newsletter-favtags into the qa-plugin folder of your server.
    - Navigate to your site, go to Admin -> Plugins and check if the plugin "Q2APRO Newsletter FavTags" is listed.
    - Change the settings of the plugin or visit the provided pages.
    - Congratulations, the plugin has been installed and your newsletter is ready!
